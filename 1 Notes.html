1. Internet Basics The Internet is a global network of computers connected to each other that allows people to share information and communicate. Important Terms 1. Website A website is a collection of related web pages. Example: www.google.com 2. Web Page A web page is a single page of a website. Example: Google Search Page 3. Browser A browser is software used to access websites. Examples: Google Chrome Mozilla Firefox Microsoft Edge Safari 4. URL (Uniform Resource Locator) A URL is the address of a website. Example: https://www.google.com https:// → Protocol www.google.com → Domain Name 2. Web Development Overview Web Development is the process of creating websites and web applications. There are three major parts: 1. Frontend Development Frontend is the part of a website that users can see and interact with. Technologies Used HTML → Structure CSS → Styling JavaScript → Interactivity Example: HTML + CSS + JavaScript ↓ Beautiful Interactive Website 2. Backend Development Backend handles the server, database, and business logic of a website. Languages Python Java PHP Node.js 3. Full Stack Development Full Stack Development is the combination of Frontend and Backend Development. 3. What is HTML? HTML stands for HyperText Markup Language. It is the standard markup language used to create the structure of web pages. HyperText HyperText refers to text that contains links to other documents or web pages. Markup Language A Markup Language uses tags to define the structure and presentation of content. HTML uses tags to tell the browser how to display content. 4. What is an HTML Element?
