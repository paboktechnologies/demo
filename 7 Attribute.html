HTML Attributes 1. What are HTML Attributes? HTML attributes provide additional information about HTML elements. Important Points All HTML elements can have attributes. Attributes provide extra information about elements. Attributes are always specified in the opening tag. Attributes usually come in name/value pairs. Syntax Content Example Visit Google Here: href → Attribute name https://www.google.com → Attribute value 2. The href Attribute The (Anchor) tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to. Syntax Link Text Example Visit Google When the user clicks the link, it opens Google. 3. The src Attribute The tag is used to display images in an HTML page. The src attribute specifies the path of the image. Syntax Example Here: img → Image tag src → Source of the image girl.jpg → Image file name 4. Types of URLs in src Attribute There are two types of URLs: 1. Absolute URL Links to an image hosted on another website. Example Characteristics: Contains complete website address. Includes protocol (https://). Used for external resources. 2. Relative URL Links to an image stored inside your website. Example 1 The image is in the same folder as the HTML file. Example 2 The image is inside the images folder. Example 3 The image path starts from the root directory of the website. 5. width and height Attributes The tag can use width and height attributes to specify the image size. Example Here: width="500" → Image width is 500 pixels height="600" → Image height is 600 pixels 6. The alt Attribute The alt attribute provides alternate text for an image. The text is displayed when: The image cannot be loaded. The internet connection is slow. Screen readers are used. Example If the image is missing, the browser displays: Girl with a jacket Why use alt? Improves accessibility. Helps search engines understand images. Provides a fallback when images fail to load. 7.
